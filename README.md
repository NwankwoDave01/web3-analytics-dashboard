# Web3 Analytics Dashboard (UI)

A modern, dark-mode **Web3 analytics dashboard UI** built with **Next.js + Tailwind CSS + Recharts**.
Designed to make on-chain style data easy to scan and understand (portfolio, allocation, performance, transactions).

## Features
- Sidebar navigation layout (Dashboard / Wallet / Tokens / Settings)
- Summary metric cards (portfolio balance, performance, activity)
- Token allocation donut chart
- Portfolio performance line chart
- Recent transactions table
- Responsive dark UI

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Recharts
- Mock data (JSON)
